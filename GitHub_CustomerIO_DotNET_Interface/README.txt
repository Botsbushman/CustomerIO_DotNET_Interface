This is just a simple .NET 4 interface written as C# Console App in order to connect and upload data via Customer.IO's REST API.  The guts of the app or juicy stuff is all in "Program.cs"

Essentially what the app is achieving is an implementation of cURL PUT Request's in .NET

IDE: Visual Studio 2010
Target Framework: .NET 4
Language: C#
Architecture: x86
